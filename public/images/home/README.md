# Tide Bridge Homepage Images

- `hero-wave.jpg`
- `about-culture.jpg`
- `business-global.jpg`
- `portfolio-kcontent.jpg`
- `portfolio-live.jpg`
- `portfolio-exhibition.jpg`
- `portfolio-brand.jpg`
- `portfolio-esports.jpg`
- `portfolio-market-entry.jpg`
- `team.jpg`
- `contact.jpg`

The homepage uses CSS background images with premium gradient and bridge/wave fallbacks, so the app builds and renders safely when these files are not present.
