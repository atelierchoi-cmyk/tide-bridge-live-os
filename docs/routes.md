# Route Map

## Public

- `/` — homepage
- `/about` — company/platform position
- `/platform` — module overview
- `/intelligence` — intelligence dashboard and subscription value layer
- `/promoters` — local promoter recruitment page
- `/agencies` — agency/IP holder page
- `/ip-catalog` — public IP preview
- `/fan-demand` — fan demand intake
- `/apply` — partner application
- `/login` — partner login placeholder

## Partner

- `/partner` — partner dashboard
- `/partner/ip-catalog` — private catalog
- `/partner/ip/[id]` — private IP detail
- `/partner/submit-offer` — structured offer intake

## Admin

- `/admin` — admin dashboard
- `/admin/ip` — IP management
- `/admin/applications` — partner application review
- `/admin/offers` — offer pipeline
- `/admin/demand` — demand dashboard
- `/admin/pnl` — P&L/BEP simulator
